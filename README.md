## Design System

- **Palette:** Warm Japandi Taupe `#F2EDE5` + Neon Coral `#FF5C35`
- **Fonts:** Plus Jakarta Sans (Display) + DM Sans (Body)
- **Features:** Lightbox-Galerie, Keyboard-Navigation, Scroll-Animationen, Portfolio-Filter
