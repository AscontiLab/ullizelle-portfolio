## 2. Website-Analyse als Marketing-Tool

### Was funktioniert
- Visuell auf Studio-Niveau
- Hero "Räume, die wirken." ist einprägsam
- 4-Schritte-Prozess baut Vertrauen auf
- Lightbox-Galerie professionell

### Kritische Lücken
- **Kontaktformular funktioniert nicht** (onsubmit="return false")
- Keine Testimonials/Social Proof
- Nur 3 von 50+ Projekten gezeigt
- Kein Blog für SEO
- Kein Analytics
- Impressum/Datenschutz fehlen

### Conversion-Pfade (SOLL)

**Pfad 1 — Direktanfrage (High Intent):**
Google → Landingpage → Case Studies → Testimonials → Formular → Bestätigung → Briefing → Angebot

**Pfad 2 — Lead Nurturing:**
Pinterest → Blog-Artikel → Stil-Finder Quiz → E-Mail Capture → Newsletter → Anfrage

**Pfad 3 — Passiv/Digital:**
Etsy → Digitales Produkt (€15-49) → Follow-up → Upsell auf Beratung
